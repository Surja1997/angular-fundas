Simple pipes that convert a text to upper case, lowercase, title case etc.

{{message | uppercase}}
{{message | lowercase}}
{{message | titlecase}}

==========================================

Pipes can also be used to convert an object to json format( the object is in the TS file )

{{person | json}}
